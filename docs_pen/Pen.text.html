# static angle(anglePoints, text, dodgeDirectionopt, offsetPixelopt)

Add a label to an angle AOB.
Parameters:
Name Type Attributes Default Description
anglePoints Array.<Array.<number>> An array [A,O,B] for the coordinates of A,O,B.
text string The string to write.
dodgeDirection number <optional>
 0 The direction to offset, given as a polar angle,relative to mid-ray of angle AOB.
offsetPixel number <optional>
 -1 The pixel distance to offset from the position. If negative, default to (text.length <= 2 ? 25 : 30).
