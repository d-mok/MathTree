# static angle(A, O, B, arcopt, radiusopt, spaceopt)

Decorate an angle AOB, always in anti-clockwise.
Parameters:
Name Type Attributes Default Description
A Array.<number> The starting point [x,y].
O Array.<number> The vertex point [x,y].
B Array.<number> The ending point [x,y].
arc number <optional>
 1 The number of arcs.
